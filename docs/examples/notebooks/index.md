# Notebooks

Jupyter notebooks for interactive systems biology workflows.

| Notebook | Topic |
|----------|-------|
| [01_search_rank.ipynb](https://github.com/MervinPraison/PraisonAIBio/blob/main/examples/notebooks/01_search_rank.ipynb) | Search and rank BioModels |

| [03_simulate_glycolysis.ipynb](https://github.com/MervinPraison/PraisonAIBio/blob/main/examples/notebooks/03_simulate_glycolysis.ipynb) | Baseline simulation (BASICO) |
| [04_perturb_compare.ipynb](https://github.com/MervinPraison/PraisonAIBio/blob/main/examples/notebooks/04_perturb_compare.ipynb) | Perturbation and comparison |
| [05_repro_bundle.ipynb](https://github.com/MervinPraison/PraisonAIBio/blob/main/examples/notebooks/05_repro_bundle.ipynb) | Repro bundle export |

```bash
python scripts/notebook_to_repro.py examples/notebooks/05_repro_bundle.ipynb
```
