# PraisonAIBio vs ClawBio

| Capability | ClawBio | PraisonAIBio |
|------------|---------|--------------|
| Domain | Genomics / clinical bioinformatics | Systems biology / SBML |
| BioModels.org | No | Core |
| ODE simulation | No | BASICO/COPASI |
| Multi-agent workflows | External LLM | Native PraisonAI |

PraisonAIBio complements ClawBio — v2 `mcp/clawbio-bridge/` connects genomics context to pathway simulation.
