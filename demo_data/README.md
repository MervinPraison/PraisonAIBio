# Demo data

Bundled slices for offline demos and benchmarks.

- `bundled_slices/` — cached metadata JSON (add via benchmarks)
- `sbml/` — optional local SBML copies for air-gapped runs
- `bundles/glycolysis_demo/` — reproducibility bundle template

Default demo model: **BIOMD0000000206** (Teusink yeast glycolysis)
