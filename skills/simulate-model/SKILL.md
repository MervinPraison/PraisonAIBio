# Simulate Model

Run BASICO time-course simulations on BioModels SBML.

## Prerequisites

```bash
pip install "praisonai-bio[simulation]"
```

## Tools

- `load_biomodel(model_id)`
- `simulate_model(model_id, duration=100)`
- `steady_state(model_id)`
- `preview_outcomes(model_id)`

## Demo model

`BIOMD0000000206` — Teusink glycolysis
